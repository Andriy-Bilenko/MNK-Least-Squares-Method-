Run version1_4.exe in this folder. Do not change anything in the current folder.
Usage:
	Paste in the prompt path to a file with data (e.g. D:\\qt projects\\MNK\\version1_2\\text1.txt (not necessarily double slashes)).
	Press the button.
Format of data in a file:
	2-3-4-6-8-9-10 2-4-3-7-8-9-12
	(X variables deliminated by '-' {space} Y variables deliminated by '-')
	(number of X variables must be equial to number of Y variables)
	(mind that there shouldn't be any '-' at the start or at the end. Program is error-prone, please check the data validity before further proceeding)
